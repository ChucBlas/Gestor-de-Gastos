## Motivación
Cansado de probar múltiples opciones dentro Play Store, encontrandome con aplicaciones cargadas de anuncios y con UX/UI poco claras y feas, decidí crear mi propia veersión sin dichas limitantes.
Para poder compartir con mis familiares y amigos, que tenían los mismos problemas y se resignaban a las opciones que habían o simplemente no usaban por "complejas".

## Acerca de
Debido a que decidí interiorizarme en el uso de Rust y nunca realicé un proyecto de este estilo en cuanto al Front, consideré que la mejor alternativa para poder realizar una aplicación multi-plataforma es Tauri 2.0, que permite adaptar el código de React de forma nativa.
